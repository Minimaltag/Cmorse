Instruções de Uso do Morse

  • - Pressione curto para ponto (.)
  • - Pressione longo para traço (-)
  • - Segure ainda mais para adicionar espaço
  • - Espere um momento para confirmar a letra
  • - Dois espaços consecutivos enviam a mensagem
  • - Use a tecla de espaço no teclado ou o botão circular
